# Minesweeper

A simple game powered by Compose Multiplatform

## Run native on MacOS
`./gradlew runDebugExecutableMacosX64` (Works on Intel processors)

## Run on iOS simulator
`./gradlew iosDeployIPhone8Debug`
`./gradlew iosDeployIPadDebug`
